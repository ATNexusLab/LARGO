# Agents & Instruções

Este índice explica onde ficam as instruções que orientam agentes (ex.: `Copilot`), e quais arquivos contém políticas e guias.

Arquivos relevantes:
- `.github/copilot-instructions.md` — instruções operacionais do Copilot (manter sincronizado)
- `.github/tasks/` — backlog e histórico de tarefas

Recomendações:
- Preferir referenciar arquivos operacionais (`.github/*`) em vez de mover.
- Criar templates de instrução para tarefas comuns e armazená-los em `docs/agents/templates/`.

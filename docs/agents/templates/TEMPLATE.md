# Template de instrução para Agents

Use este template para criar instruções reutilizáveis para agents (ex.: Copilot, automações internas).

Exemplo mínimo:

- Objetivo: Tarefa a ser executada pelo agent
- Entradas: descrição, dados contextuais
- Resposta esperada: formato JSON ou instruções passo-a-passo

Salvar templates em `docs/agents/templates/` e referenciar no `docs/agents/index.md`.

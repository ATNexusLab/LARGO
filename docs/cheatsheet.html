◆ Fluxo Principal — Projeto do Zero
0
/gsd:map-codebase
Apenas se já tiver código. Agentes paralelos analisam stack, arquitetura e padrões antes de iniciar.
1
/gsd:new-project
Faz perguntas → pesquisa domínio → extrai requisitos → gera roadmap por fases. Você aprova o roadmap.
Cria: PROJECT.mdREQUIREMENTS.mdROADMAP.mdSTATE.md
2
/gsd:discuss-phase N
Captura suas decisões antes de qualquer planejamento — layout, interações, comportamento. Sem isso, gera defaults razoáveis.
Cria: N-CONTEXT.md
3
/gsd:plan-phase N
Pesquisa como implementar → cria 2–3 planos atômicos XML → verifica planos contra requisitos. Loop até passar.
Cria: N-RESEARCH.mdN-1-PLAN.mdN-2-PLAN.md
4
/gsd:execute-phase N
Roda planos em waves paralelas. Contexto fresco (200k tokens) por plano. Commit atômico por tarefa. Verifica entrega.
Cria: N-N-SUMMARY.mdN-VERIFICATION.md
5
/gsd:verify-work N
Você testa manualmente. Sistema extrai entregáveis, guia passo a passo. Falhas → diagnóstico automático + planos de fix.
Cria: N-UAT.mdfix plans (se falhar)
6
/gsd:ship N
Cria PR do trabalho verificado com body gerado automaticamente. --draft para PR em rascunho.
↻
/gsd:next
Auto-detecta e executa o próximo passo. Para quem quer delegar a sequência por completo.
Loop por fase: discuss → plan → execute → verify → ship. Repita para cada fase. Ao terminar todas: /gsd:complete-milestone → /gsd:new-milestone

Dicas de Uso Avançado
Modos de Discuss
discuss (padrão)
Claude te entrevista. Você toma decisões explícitas sobre cada área ambígua da fase.
assumptions
Claude analisa o código, mostra o que faria e por quê. Você corrige só o que está errado.
Seeds — Ideias Futuras com Gatilho
Use /gsd:plant-seed para capturar ideias com condição de disparo. Ex: "adicionar dark mode quando tiver design system estável". O sistema as apresenta no milestone correto — sem poluir o backlog atual.
Context Rot — O Problema que GSD Resolve
À medida que Claude enche o contexto, a qualidade degrada. GSD força contexto fresco por plano via subagentes, mantendo qualidade constante do início ao fim da fase.

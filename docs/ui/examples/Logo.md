# Logo

Componente Logo — carrega a imagem do logotipo.

Uso
```tsx
import Logo from './react/Logo';

<Logo src="/docs/design/logo.png" alt="Logo" />
```

Observações
- Você mencionou que irá fornecer SVG e variações. Coloque as versões finais em `docs/design/assets/` ou `docs/design/` e atualize o src conforme necessário.

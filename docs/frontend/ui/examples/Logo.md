# Logo

Componente Logo — carrega a imagem do logotipo.

Uso
```tsx
import Logo from './react/Logo';

<Logo src="/docs/design/assets/logo.png" alt="Logo" />
```

Observações
- Logo disponível em `docs/design/assets/logo.png`. Quando variações SVG estiverem prontas, salvar como `logo-horizontal.svg`, `logo-vertical.svg`, `logo-mark.svg` no mesmo diretório.

# Brand Guidelines — Identidade Visual (inicial)

Essência da marca
- Tom: profissional, confiável e simples. Visual limpo, foco em dados e legibilidade.
- Público: usuários de gestão financeira / tarefas.

Logo
- Logo disponível em `docs/design/assets/logo.png`. Versões SVG planejadas: `logo-horizontal.svg`, `logo-vertical.svg`, `logo-mark.svg` e favicons no mesmo diretório.
- Forneça versões em cor principal, negativo (branco) e mono.

Uso de cores
- Cor primária (accent): usar para CTAs, ações primárias e gráficos importantes.
- Neutros: para fundos, texto secundário e bordas.
- Evitar usar mais que duas cores de destaque por tela.

Tipografia
- Preferência por fontes sans-serif modernas (ex.: Inter, Roboto ou system sans). Confirmar escolha para aplicar tokens.
- Hierarquia clara: H1 > H2 > H3 > body; manter espaçamentos consistentes.

Acessibilidade
- Objetivo: mínimo WCAG AA para texto normal (contraste 4.5:1). Para textos grandes, mínimo 3:1.
- Testar contrastes das combinações primárias/foreground.

Elementos visuais
- Radius: 8px (padrão) — variantes menores para componentes compactos.
- Sombra: leve para elevar cards e modais, evitar sombras fortes.
- Ícones: estilo stroke/duotone consistente; preferir biblioteca (Heroicons) ajustada ao token radius.

Fotografia / Ilustrações
- Estilo limpo, preferência por imagens com fundo claro e tratamento de cor neutra.

Padrões de aplicação
- Botões: primary (filled), secondary (outline/ghost), danger (semantic).
- Formulários: labels acima, espaçamento consistente e estado de erro claro.

Entrega esperada
- Quando enviar o logo, eu atualizo tokens, exemplos e gerei variantes (SVG/PNG) nas pastas corretas.

Notas finais
- Este documento é o ponto de partida; ajustes serão feitos após a sua escolha de fonte e tom final da paleta.
